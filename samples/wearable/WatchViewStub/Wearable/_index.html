This sample demonstrates how to specify different layouts for round and rectangular screens. The layouts used in this sample are as follows:

  • Round: The background is a blue/green ring surrounding red text.
  • Rect: The background is a blue/breen rounded rectangle with red text on top.
Additionally, a unique animation is played when the user clicks on the layout, depending on whether the screen is round or rectangular:
  • Round: The layout rotates 360 degrees.
  • Rect: The layout shrinks to 70% and then grows back to 100%.
